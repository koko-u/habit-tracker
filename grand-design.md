# 習慣を継続するアプリケーション Habit Tracker

## 基本機能

- 継続したい習慣を登録する
- 日々、習慣を実行したか、しなかったかを記録する

## 追加機能

1. 過去の履歴をカレンダーとして表示する
2. 習慣が継続していることを可視化する
3. 習慣を実行した／しなかったの 2値ではなく、より詳しい記録を残せる
4. 年間、あるいは月間の達成率をグラフで表示する

# 技術スタック

## バックエンド

- Rust
- Actix-Web
- PostgreSQL

## フロントエンド

- Angular

## 認証

- JWT
- ログインして自分のハビットを管理するのみ
